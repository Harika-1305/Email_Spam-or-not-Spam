# 📧 Spam Email Detection using Naive Bayes

## 📌 Project Overview
This project builds a machine learning model to classify emails as spam or non-spam (ham). It uses the Naive Bayes algorithm, which is highly effective for text classification tasks.

## 📊 Dataset
- Source: UCI Machine Learning Repository  
- Dataset: Spambase Dataset  
- Features: Word frequencies and email characteristics  
- Target Variable:  
  - 0 → Non-Spam (Ham)  
  - 1 → Spam  

## ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

## 🚀 Steps Performed
1. Imported required libraries  
2. Loaded dataset using Pandas  
3. Handled missing values  
4. Split data into features (X) and target (y)  
5. Performed train-test split  
6. Applied Naive Bayes (MultinomialNB) model  
7. Trained the model  
8. Predicted results on test data  
9. Evaluated model using:
   - Accuracy  
   - Confusion Matrix  
   - Classification Report  
10. Visualized actual vs predicted values  

## 🤖 Model Details
- Algorithm: Multinomial Naive Bayes  
- Suitable for: Text classification and frequency-based data
